# Actividad figuras con cartas en Three.js

Se creó una página web con figuras 3D rotando en un banner superior y tarjetas de las figuras.

## Instrucciones de uso
1. Clona el repositorio: `https://github.com/gabs147/Three.js-Figuras`
2. Abre el proyecto en tu editor de código.
3. Para ejecutar la pagina abre el archivo `index.html` en tu navegador favorito.

## Tecnologías utilizadas
- HTML5
- Three.js
- CSS
- Bootstrap
